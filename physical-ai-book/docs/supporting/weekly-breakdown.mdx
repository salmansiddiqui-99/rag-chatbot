# Weekly Breakdown

This course spans **13 weeks** with ~6-8 hours of study time per week (mix of reading, labs, and projects).

---

## Week 1: Introduction to Physical AI

**Module**: [Introduction](../introduction.mdx)

**Topics**:

- What is Physical AI? Embodied intelligence vs. digital AI
- The humanoid robotics landscape (Boston Dynamics, Tesla Optimus, Figure)
- Course roadmap and learning objectives
- Physical AI stack overview

**Deliverables**:

- Set up ROS 2 Humble development environment
- Complete "Hello World" ROS 2 publisher-subscriber node
- Install Gazebo Classic 11

**Resources**:

- [Introduction to Physical AI](../introduction.mdx)
- [Learning Outcomes](./learning-outcomes.mdx)
- [ROS 2 Installation Guide](https://docs.ros.org/en/humble/Installation.html)

---

## Week 2: ROS 2 Fundamentals

**Module**: [Module 1 - ROS 2](../module-1-ros2/index.mdx)

**Topics**:

- ROS 2 architecture and DDS middleware
- Nodes, topics, publishers, and subscribers
- ROS 2 command-line tools (`ros2 node`, `ros2 topic`, `ros2 service`)
- Quality of Service (QoS) policies

**Deliverables**:

- **Lab 1**: Create a publisher-subscriber pair for simulated sensor data
- **Lab 2**: Implement a simple ROS 2 service for robot configuration

**Resources**:

- [ROS 2 Architecture](../module-1-ros2/01-architecture.mdx)
- [Nodes, Topics, Services](../module-1-ros2/02-nodes-topics-services.mdx)

---

## Week 3: rclpy Development & URDF Modeling

**Module**: [Module 1 - ROS 2](../module-1-ros2/index.mdx)

**Topics**:

- Python ROS 2 nodes with rclpy
- Timers, parameters, and logging
- URDF syntax for robot descriptions
- Defining joints, links, and kinematic chains

**Deliverables**:

- **Lab 3**: Build a configurable ROS 2 node with parameters
- Create URDF model for a simple 2-DOF arm

**Resources**:

- [rclpy Integration](../module-1-ros2/03-rclpy-integration.mdx)
- [URDF for Humanoids](../module-1-ros2/04-urdf-humanoids.mdx)

- **Reading**: 20 hours total (~1.5 hours/week)

- **Total**: ~82 hours over 13 weeks (~6-8 hours/week)
