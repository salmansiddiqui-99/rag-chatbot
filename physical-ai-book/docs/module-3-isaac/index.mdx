# Module 3: The AI-Robot Brain (NVIDIA Isaac)

## Overview

NVIDIA Isaac is an end-to-end platform for AI-powered robotics, combining:

- **Isaac Sim**: Photorealistic simulation with GPU-accelerated physics (PhysX)
- **Isaac ROS**: Pre-built AI perception pipelines (VSLAM, object detection, pose estimation)
- **Isaac SDK**: Low-level robot control and sensor drivers (deprecated, migrated to ROS 2)

This module focuses on Isaac Sim for **synthetic data generation** and Isaac ROS for **real-time perception** on NVIDIA Jetson hardware.

---

## What You'll Learn

By the end of this module, you will:

- ✅ **Generate Synthetic Data**: Create millions of labeled images for training perception models
- ✅ **Implement VSLAM**: Visual Simultaneous Localization and Mapping for navigation
- ✅ **Integrate Isaac ROS**: Use GPU-accelerated perception nodes on Jetson Orin
- ✅ **Adapt Nav2**: Configure navigation stacks for bipedal humanoid robots
